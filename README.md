<h2 align="right">1.1. INTRODUCTION 📰</h2>

O Sistema de Inventário de Itens é uma aplicação simples e eficiente projetada para ajudar pequenas e médias empresas a gerenciar seus estoques de maneira organizada e precisa. Através deste sistema, os usuários podem acompanhar a entrada, saída e o saldo de itens em tempo real, facilitando o controle e a tomada de decisões.
<h2 align="right">1.2. DESCRIPTION 📝</h2>

O Sistema de Inventário de Itens é uma ferramenta indispensável para empresas que desejam manter um controle rigoroso e eficiente dos seus estoques. Projetado com foco na simplicidade e funcionalidade, este sistema facilita a gestão de entradas, saídas e níveis de estoque, proporcionando uma visão clara e detalhada dos itens armazenados.
